# A draggable, sortable list example using React-Motion and React-DnD

![alt tag](https://github.com/xbrightmatterx/ReactAnimatedList/blob/master/ListDemo.gif)

## How To Start

- clone / download this repo
- run `npm install`
- run `npm start`
- navigate to `localhost:8080`

## Tools Used

- Styled-Components
- React
- React-Motion
- React-DnD
- LoDash
- Babel
- Webpack
